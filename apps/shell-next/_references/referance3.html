Architecture Overview

This portfolio is built as a static site using Next.js 14 with App Router. The decision to use Next.js over a simple Vite + React SPA was driven by the need for better SEO capabilities and the ability to leverage React Server Components (RSC) for heavier content sections.

State Management

Local State (React Hooks)

Used for UI-specific interactions like modal visibility, form inputs, and animations. Kept strictly collocated with the components that need it to prevent prop drilling.

Server State

Leveraging React Server Components means most data is fetched on the server and passed down. No client-side global store (Redux/Zustand) was necessary for this scale.

info

Trade-off: While client-side caching libraries like React Query are powerful, for a static portfolio, build-time data fetching (SSG) provides superior Time to First Byte (TTFB).
